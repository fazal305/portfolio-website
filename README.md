# Portfolio Website

A futuristic dark-themed personal portfolio built with HTML, CSS, JavaScript, Bootstrap 5, jQuery and Firebase Hosting.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery
- Firebase Firestore
- Firebase Hosting

## Features

- Responsive design
- Animated UI
- Typewriter hero section
- Scroll reveal animations
- Firebase contact form
- Project filtering
- Bootstrap layouts
- Desktop & scripting project showcase
